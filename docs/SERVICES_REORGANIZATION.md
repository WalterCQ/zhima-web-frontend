# 知马同城板块重组说明（马来西亚本地化）

## 更新日期
2025-10-14

## 概述
按照新需求重新组织知马同城页面，将服务分为六大板块：车、房、工作、社交、服务、生活。
同时进行马来西亚本地化适配，保留并扩展细分内容，维持约280+个可点击链接。

### 2. 页面布局

保持现有三列 + 右侧边栏布局：
- **第一列**：车 + 房
- **第二列**：工作 + 社交
- **第三列**：服务 + 生活
- **右侧边栏**：热门服务（保持不变）

### 3. 路由跳转逻辑

#### 带*服务项
所有带*号的服务项点击后跳转到 `ServicesList` 页面，传递 `category` 参数。

#### 专属页面项
- **车板块**：二手车 → `CarsList`；租车 → `ServicesList`
- **房板块**：二手房 → `HousingSecondhand`；新房 → `HousingNew`；租房 → `RentList`；其他 → `HousingHome`
- **工作板块**：全部 → `JobsList`（兼职工作带 type 参数）
- **社交板块**：全部 → `Circles`（知马圈）
- **生活板块**：二手家具 → `ServicesList` (secondhand group)
